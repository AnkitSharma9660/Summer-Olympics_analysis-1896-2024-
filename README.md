# Summer-Olympics_analysis-1896-2024-
# 🏅 Summer Olympics Data Analytics Dashboard (1896–2024)

## Project Overview

This project is an interactive **data analytics dashboard** that explores historical **Summer Olympics data from 1896 to 2024**.
It helps users analyse trends in athlete participation, medal distribution, country performance, and event growth using visualizations.

## Features

* Year-wise medal tally analysis
* Country-wise Olympic performance insights
* Athlete participation trends over time
* Sport-wise medal distribution
* Interactive filters and dynamic charts

## Tech Stack

* **Python**
* **Pandas, NumPy**
* **Matplotlib, Seaborn, Plotly**
* **Streamlit (for dashboard deployment)**

## 🌐 Live Demo

👉 *[https://summer-olympic-analysis.streamlit.app/]*

## 📁 Dataset

Historical Summer Olympics dataset including athlete details, events, medals, and participating nations.

## 🎯 Objective

To perform exploratory data analysis and build an **interactive web dashboard** that provides meaningful insights into Olympic history.
